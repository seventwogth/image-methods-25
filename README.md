## ЛР1. Вычисление освещенности в точках поверхности

Реализация: `first/main.py`.

Вычисляется цветная освещенность точки на плоскости треугольника от одного точечного источника света.

### Формулы ЛР1

1. Сила излучения источника с учетом диаграммы направленности:

\[
I(RGB, \vec s) = I_0(RGB) \cdot \max(0, \cos\theta),
\quad
\cos\theta = \frac{\vec s\cdot\vec O}{\|\vec s\|\,\|\vec O\|}
\]

2. Освещенность точки:

\[
E(RGB, \vec P_T) = \frac{I(RGB, \vec s)\cdot\max(0,\cos\alpha)}{R^2},
\quad
\cos\alpha = \frac{\vec s\cdot\vec N}{\|\vec s\|\,\|\vec N\|},
\quad
R^2=\|\vec s\|^2
\]

3. Перевод локальных координат в глобальные:

\[
\vec P_T = \vec P_0 + \hat e_1\,x + \hat e_2\,y,
\quad
\hat e_1 = \frac{\vec P_1-\vec P_0}{\|\vec P_1-\vec P_0\|},
\quad
\hat e_2 = \frac{\vec P_2-\vec P_0}{\|\vec P_2-\vec P_0\|}
\]

4. Нормаль плоскости:

\[
\vec N = \frac{(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)}{\|(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)\|}
\]

## ЛР2. Вычисление яркости в точках на поверхности

Реализация: `second/main.py`.

В этой работе рассматриваются **два точечных источника**, освещенность от каждого источника и итоговая яркость точки с учетом диффузно-зеркальной модели отражения (BRDF).

### Формулы ЛР2

1. Для каждого источника `i` сначала считается освещенность:

\[
E_i(RGB, \vec P_T) = \frac{I_i(RGB, \vec s_i)\cdot\max(0,\cos\alpha_i)}{R_i^2}
\]

где

\[
I_i(RGB, \vec s_i) = I_{0,i}(RGB)\cdot\max(0,\cos\theta_i)
\]

2. Затем яркость точки:

\[
L(RGB, \vec P_T, \vec v) = \frac{1}{\pi}\sum_i E_i(RGB, \vec P_T)\cdot f(RGB, \vec P_T, \vec v, \vec s_i)
\]

3. Двунаправленная функция отражения (BRDF):

\[
f(RGB, \vec P_T, \vec v, \vec s_i) = K(RGB)\cdot\left[k_d + k_s\cdot(\max(0,\vec h_i\cdot\vec N))^{k_e}\right]
\]

4. Полувектор:

\[
\vec h_i = \frac{\vec v + \vec s_i}{\|\vec v + \vec s_i\|}
\]

где:

- `K(RGB)` — цвет поверхности;
- `k_d` — коэффициент диффузного отражения;
- `k_s` — коэффициент зеркального отражения;
- `k_e` — показатель степени (ширина блика);
- `v` — направление наблюдения;
- `N` — нормаль к поверхности;
- `s_i` — направление на источник света.

### Алгоритм ЛР2

1. Задаются два источника (`I0`, `O`, `P_L`), геометрия плоскости (`P0`, `P1`, `P2`) и параметры материала (`K`, `k_d`, `k_s`, `k_e`).
2. Для каждой локальной точки `(x, y)` вычисляется глобальная точка `P_T`.
3. Для каждого источника считается `E_i(RGB, P_T)`.
4. Для каждого источника считается BRDF-фактор `f(...)` через полувектор `h_i`.
5. По сумме вкладов вычисляется итоговая яркость `L(RGB, P_T, v)`.
6. В консоль выводятся три таблицы: `E1`, `E2` и `L`.

# ЛР 1. Вычисление освещенности в точках поверхности

В репозитории реализована программа на Python для расчета «цветной» освещенности точек на плоскости треугольника от точечного источника света с направленной диаграммой излучения.

## Структура

- `first/main.py` — основная программа, которая:
  - задает входные параметры источника света и треугольной площадки;
  - переводит локальные координаты `(x, y)` в глобальные координаты точки `P_T`;
  - вычисляет нормаль к плоскости;
  - считает освещенность `E(RGB, P_T)` в сетке точек;
  - выводит таблицу значений.

## Используемые формулы

Пусть:

- `I0(RGB)` — цветовая сила излучения источника по его оси;
- `O` — единичный вектор оси источника;
- `P_L` — положение источника света;
- `P0, P1, P2` — вершины треугольной площадки;
- `P_T` — точка на плоскости, в которой считаем освещенность.

### 1) Сила излучения под углом к оси источника

\[
I(RGB, \vec s) = I_0(RGB) \cdot \cos\theta
\]

где

\[
\cos\theta = \frac{\vec s \cdot \vec O}{\|\vec s\|\,\|\vec O\|}
\]

В программе отрицательные значения \(\cos\theta\) ограничиваются нулем (источник не излучает «назад»):

\[
\cos\theta = \max(0, \cos\theta)
\]

### 2) Цветная освещенность точки

\[
E(RGB, \vec P_T) = \frac{I(RGB, \vec s) \cdot \cos\alpha}{R^2}
\]

где

\[
\cos\alpha = \frac{\vec s \cdot \vec N}{\|\vec s\|\,\|\vec N\|},
\qquad
R^2 = \|\vec s\|^2,
\qquad
\vec s = \vec P_T - \vec P_L
\]

Также используется ограничение \(\cos\alpha \ge 0\), чтобы учитывать только освещение лицевой стороны поверхности.

### 3) Перевод локальных координат точки в глобальные

Для локальных координат `x, y`:

\[
\vec P_T = \vec P_0 + \hat e_1\,x + \hat e_2\,y
\]

где

\[
\hat e_1 = \frac{\vec P_1 - \vec P_0}{\|\vec P_1 - \vec P_0\|},
\qquad
\hat e_2 = \frac{\vec P_2 - \vec P_0}{\|\vec P_2 - \vec P_0\|}
\]

### 4) Нормаль к плоскости треугольника

\[
\vec N = \frac{(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)}{\|(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)\|}
\]

## Шаги алгоритма

1. Задать параметры источника: `I0(RGB)`, ось `O`, позицию `P_L`.
2. Задать треугольную площадку: `P0`, `P1`, `P2`.
3. Для каждой пары локальных координат `(x, y)`:
   - вычислить глобальную точку `P_T`;
   - построить вектор `s = P_T - P_L`;
   - вычислить `cos(theta)` и получить `I(RGB, s)`;
   - вычислить нормаль `N`, `cos(alpha)` и расстояние `R^2`;
   - получить `E(RGB, P_T)`.
4. Вывести таблицу освещенности для всей сетки точек.
