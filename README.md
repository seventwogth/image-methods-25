# ЛР 1. Вычисление освещенности в точках поверхности

В репозитории реализована программа на Python для расчета «цветной» освещенности точек на плоскости треугольника от точечного источника света с направленной диаграммой излучения.

## Структура

- `first/main.py` — основная программа, которая:
  - задает входные параметры источника света и треугольной площадки;
  - переводит локальные координаты `(x, y)` в глобальные координаты точки `P_T`;
  - вычисляет нормаль к плоскости;
  - считает освещенность `E(RGB, P_T)` в сетке точек;
  - выводит таблицу значений.

## Используемые формулы

Пусть:

- `I0(RGB)` — цветовая сила излучения источника по его оси;
- `O` — единичный вектор оси источника;
- `P_L` — положение источника света;
- `P0, P1, P2` — вершины треугольной площадки;
- `P_T` — точка на плоскости, в которой считаем освещенность.

### 1) Сила излучения под углом к оси источника

\[
I(RGB, \vec s) = I_0(RGB) \cdot \cos\theta
\]

где

\[
\cos\theta = \frac{\vec s \cdot \vec O}{\|\vec s\|\,\|\vec O\|}
\]

В программе отрицательные значения \(\cos\theta\) ограничиваются нулем (источник не излучает «назад»):

\[
\cos\theta = \max(0, \cos\theta)
\]

### 2) Цветная освещенность точки

\[
E(RGB, \vec P_T) = \frac{I(RGB, \vec s) \cdot \cos\alpha}{R^2}
\]

где

\[
\cos\alpha = \frac{\vec s \cdot \vec N}{\|\vec s\|\,\|\vec N\|},
\qquad
R^2 = \|\vec s\|^2,
\qquad
\vec s = \vec P_T - \vec P_L
\]

Также используется ограничение \(\cos\alpha \ge 0\), чтобы учитывать только освещение лицевой стороны поверхности.

### 3) Перевод локальных координат точки в глобальные

Для локальных координат `x, y`:

\[
\vec P_T = \vec P_0 + \hat e_1\,x + \hat e_2\,y
\]

где

\[
\hat e_1 = \frac{\vec P_1 - \vec P_0}{\|\vec P_1 - \vec P_0\|},
\qquad
\hat e_2 = \frac{\vec P_2 - \vec P_0}{\|\vec P_2 - \vec P_0\|}
\]

### 4) Нормаль к плоскости треугольника

\[
\vec N = \frac{(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)}{\|(\vec P_2-\vec P_0)\times(\vec P_1-\vec P_0)\|}
\]

## Шаги алгоритма

1. Задать параметры источника: `I0(RGB)`, ось `O`, позицию `P_L`.
2. Задать треугольную площадку: `P0`, `P1`, `P2`.
3. Для каждой пары локальных координат `(x, y)`:
   - вычислить глобальную точку `P_T`;
   - построить вектор `s = P_T - P_L`;
   - вычислить `cos(theta)` и получить `I(RGB, s)`;
   - вычислить нормаль `N`, `cos(alpha)` и расстояние `R^2`;
   - получить `E(RGB, P_T)`.
4. Вывести таблицу освещенности для всей сетки точек.

## Запуск

```bash
python3 first/main.py
```

В консоль выводится таблица вида `y\x -> (R, G, B)`.

## Пример входных данных (вшит в `main.py`)

- `I0(RGB) = (1200, 900, 700)`
- `O = (0.3, -0.9, -0.3)`
- `P_L = (2, 5, 4)`
- `P0 = (0, 0, 0)`
- `P1 = (4, 1, 0)`
- `P2 = (1, 3.5, 0)`
- Сетка локальных координат:
  - `x = [0.0, 0.5, 1.0, 1.5, 2.0]`
  - `y = [0.0, 0.4, 0.8, 1.2, 1.6]`

При необходимости значения можно изменить прямо в функции `main()`.
