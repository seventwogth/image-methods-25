## Лабораторная работа 1: Расчёт освещённости на треугольной пластине

### Цель работы

Изучить, как вычисляется освещённость в точках на плоскости треугольника при освещении его точечным источником света с заданной диаграммой излучения.

### Используемые формулы

**1. Цветная сила излучения источника:**
$$I(RGB, s) = I_0(RGB) \cdot \cos\theta$$

где:
- $I_0(RGB)$ — цветная сила излучения источника в направлении оси
- $\vec{O}$ — ось источника света, $\|\vec{O}\| = 1$
- $\theta$ — угол между направлением распространения света и осью источника
- $\cos\theta$ — диаграмма излучения

**2. Нормаль плоскости треугольника:**
$$\vec{N} = \frac{(P_2 - P_0) \times (P_1 - P_0)}{\|(P_1 - P_0) \times (P_2 - P_0)\|}$$

**3. Перевод локальных координат в глобальные:**
$$\vec{P_T} = \vec{P_0} + \frac{P_1 - P_0}{\|P_1 - P_0\|}x + \frac{P_2 - P_0}{\|P_2 - P_0\|}y$$

где $x, y$ — смещения вдоль рёбер треугольника.

**4. Вектор до источника света:**
$$s = \vec{P_T} - \vec{P_L}$$
$$R^2 = \|s\|^2$$

**5. Цветная освещённость точки:**
$$E(RGB, \vec{P_T}) = \frac{I(RGB, s) \cdot \cos\alpha}{R^2}$$

где:
- $\cos\alpha = \frac{s \cdot \vec{N}}{\|s\|}$ — угол между направлением света и нормалью поверхности
- $R$ — расстояние от источника света до точки

### Алгоритм

1. Задать параметры источника света
2. Задать координаты вершин треугольника
3. Вычислить нормаль поверхности
4. Перевести локальные координаты точек в глобальные
5. Найти вектор до источника света
6. Вычислить углы между векторами
7. Рассчитать освещённость точки

### Описание программы

Программа `first/main.py` реализует описанный алгоритм:
1. Задаёт источник света (позиция $P_L$, ось $O$, RGB-интенсивность $I_0$)
2. Задаёт треугольную пластину (координаты вершин $P_0, P_1, P_2$)
3. Строит сетку точек на плоскости через локальные координаты $(x, y)$
4. Для каждой точки вычисляет освещённость по формулам выше
5. Выводит результаты в виде таблицы

| `TrianglePlane` | Класс-контейнер для хранения координат трёх вершин треугольника: `p0`, `p1`, `p2` |

## Лабораторная работа 2: Вычисление яркости в точках на поверхности

### Цель работы

Изучить, как вычисляется яркость в точках на плоскости треугольника с учётом освещения от точечных источников света и диффузно-отражательных свойств поверхности.

### Используемые формулы

Формулы из лабораторной работы 1 (освещённость, нормаль, координаты) используются здесь повторно.

**1. Цветная сила излучения источника:**
$$I(RGB, s) = I_0(RGB) \cdot \cos\theta$$

**2. Цветная освещённость точки:**
$$E(RGB, \vec{P_T}) = \frac{I(RGB, s) \cdot \cos\alpha}{R^2}$$

**3. Нормаль плоскости:**
$$\vec{N} = \frac{(P_2 - P_0) \times (P_1 - P_0)}{\|(P_1 - P_0) \times (P_2 - P_0)\|}$$

**4. Перевод локальных координат:**
$$\vec{P_T} = \vec{P_0} + \frac{P_1 - P_0}{\|P_1 - P_0\|}x + \frac{P_2 - P_0}{\|P_2 - P_0\|}y$$

**5. Вектор от точки до источника:**
$$s = \vec{P_T} - \vec{P_L}$$

---

### Двунаправленная функция отражения (BRDF)

$$f_r = k_d + k_s(\vec{h} \cdot \vec{n})^m$$

где:
- $k_d$ — коэффициент диффузного отражения поверхности
- $k_s$ — коэффициент зеркального отражения поверхности
- $m$ — параметр, определяющий ширину блика
- $\vec{h}$ — средний вектор (half vector)
- $\vec{n}$ — нормаль к поверхности

### Средний вектор (half vector)

$$\vec{h} = \frac{\vec{l} + \vec{v}}{\|\vec{l} + \vec{v}\|}$$

где:
- $\vec{l}$ — направление на источник света
- $\vec{v}$ — направление наблюдения

### Яркость точки

$$L = \sum_i E_i \cdot f_r(\vec{x}, \vec{v}, \vec{l}) \cdot \rho$$

где:
- $E_i$ — освещённость от $i$-го источника света
- $\vec{x}$ — координаты точки
- $\vec{v}$ — направление наблюдения
- $\vec{l}$ — направление на источник света
- $f_r$ — BRDF
- $\rho$ — цвет поверхности

### Описание программы

Программа `second/main.py` реализует:
1. Задаёт два источника света с различными параметрами
2. Задаёт треугольную пластину и материал (цвет $\rho$, коэффициенты $k_d$, $k_s$, параметр $m$)
3. Задаёт направление на наблюдателя $\vec{v}$
4. Для каждой точки вычисляет:
   - $E_1(RGB)$ — освещённость от первого источника
   - $E_2(RGB)$ — освещённость от второго источника
   - $L(RGB)$ — итоговая яркость с учётом BRDF
5. Выводит три таблицы результатов

### Описание функций

| Функция / Класс | Назначение |
|-----------------|------------|
| `LightSource` | Класс-контейнер для хранения параметров источника света: `name` (имя), `intensity_rgb` (сила излучения $I_0$), `axis` (ось $O$), `position` (позиция $P_L$) |
| `TrianglePlane` | Класс-контейнер для хранения координат трёх вершин треугольника: `p0`, `p1`, `p2` |
| `Material` | Класс-контейнер для хранения оптических свойств материала: `color_rgb` (цвет $\rho$), `kd` (коэффициент диффузного отражения), `ks` (коэффициент зеркального отражения), `ke` (параметр блеска $m$) |
| `dot(a, b)` | Скалярное произведение векторов $a \cdot b$ |
| `add(a, b)` | Сложение векторов $a + b$ |
| `sub(a, b)` | Вычитание векторов $a - b$ |
| `mul(v, k)` | Умножение вектора на скаляр $k \cdot v$ |
| `cross(a, b)` | Векторное произведение $a \times b$ |
| `norm(v)` | Вычисление длины (нормы) вектора $\|v\|$ |
| `normalize(v)` | Нормализация вектора (приведение к единичной длине) |
| `clamp_nonnegative(x)` | Ограничение значения $x$ снизу нулём |
| `to_global_point(plane, x, y)` | Перевод локальных координат $(x, y)$ в глобальные |
| `plane_normal(plane)` | Вычисление нормали треугольной плоскости |
| `radiation_intensity(light, direction_from_light)` | Вычисление силы излучения в направлении с учётом косинусной диаграммы |
| `illuminance_from_light(light, plane, point)` | Вычисление освещённости точки от одного источника |
| `brdf(material, n, v, s_to_light)` | Вычисление BRDF: $f_r = k_d + k_s(\vec{h} \cdot \vec{n})^{m}$, где $\vec{h} = \frac{\vec{v} + \vec{s}}{\|\vec{v} + \vec{s}\|}$ |
| `brightness_at_point(...)` | Вычисление итоговой яркости $L$ с учётом всех источников по формуле $L = \sum E_i \cdot f_r \cdot \rho / \pi$ |
| `print_table(...)` | Вывод таблицы в консоль |

### Генерируемые CSV-таблицы

- `second/e1_table.csv` — значения освещённости $E_1(RGB, P_T)$ для точек, заданных локальными координатами
- `second/e2_table.csv` — значения освещённости $E_2(RGB, P_T)$ для тех же точек
- `second/l_table.csv` — значения яркости $L(RGB, P_T, v)$ для тех же точек на плоскости

## Лабораторная работа 3-1: Вычисление яркости в заданном направлении

### Цель работы

Изучить, как вычисляется яркость в заданных направлениях с учетом отражения от зеркальной поверхности.

### Используемые формулы

В работе используются формулы из лабораторной работы 2 для силы излучения и освещённости точки, а также формула зеркального отражения.

**1. Цветная сила излучения источника:**
$$I(RGB, s) = I_0(RGB) \cdot \cos\theta$$

**2. Цветная освещённость точки:**
$$E(RGB, \vec{P_T}) = \frac{I(RGB, s) \cdot \cos\alpha}{R^2}$$

**3. Формула зеркального отражения:**
$$\vec{O'} = \vec{O} - 2(\vec{O} \cdot \vec{N})\vec{N}$$

где:
- $\vec{O'}$ — отражённый луч
- $\vec{O}$ — падающий луч
- $\vec{N}$ — нормаль к зеркальной поверхности

**4. Положения наблюдателя:**

В постановке используются положение наблюдателя $P_0$ и положение виртуального наблюдателя $P_0'$.

**Принятая в реализации интерпретация яркости:**

Так как в `docs` для ЛР_3-1 явно заданы входные данные $K(RGB)$ и $k_d$, но не заданы параметры $k_s$ и $m$, в текущей реализации итоговая яркость для заданного направления наблюдения вычисляется как произведение освещённости точки на цвет поверхности и коэффициент $k_d$, если направление наблюдения совпадает с отражённым лучом. Для остальных направлений яркость принимается равной нулю.

### Шаги алгоритма

1. Задать входные данные источника света: $I_{01}(RGB)$, $\vec{O_1}$, $\vec{P_L}$
2. Задать положения наблюдателя $\vec{P_{01}}, \vec{P_{02}}, \vec{P_{03}}, \vec{P_{04}}, \vec{P_{05}}$
3. Задать параметры поверхности: $K(RGB)$ и $k_d$
4. Вычислить силу излучения источника в направлении точки отражения
5. Вычислить освещённость точки зеркальной поверхности
6. Вычислить отражённый луч по формуле зеркального отражения
7. Для каждого наблюдателя вычислить положение виртуального наблюдателя $P_0'$
8. Для каждого направления наблюдения вычислить яркость $L(RGB, P_T, v)$

### Описание программы

Программа `third/main.py` реализует:
1. Задаёт один точечный источник света с цветной интенсивностью, осью излучения и положением
2. Задаёт зеркальную поверхность одной отражающей точкой $P_T$, нормалью, цветом поверхности и коэффициентом $k_d$
3. Задаёт пять положений наблюдателя $P01 \dots P05$
4. Для каждого наблюдателя вычисляет положение виртуального наблюдателя, направление отражённого луча и итоговую яркость
5. Выводит входные данные и таблицу результатов в консоль
6. Сохраняет входные данные и результаты в отдельные CSV-файлы внутри директории `third`
